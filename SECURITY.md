This repository is a sanitized lab project.

## Do Not Commit

- Real public IP addresses
- Passwords
- API keys
- Firewall backup exports
- VPN pre-shared keys
- Production hostnames
- Organization-specific network maps
- Sensitive screenshots

## Safe Sharing Rules

- Use documentation examples instead of full production exports.
- Replace sensitive values with placeholders.
- Use RFC 5737 documentation IP ranges or private lab ranges.
- Review every file before publishing.
